🔑 API Key Setup

📺 YouTube

  1. Google Cloud Console
  2. Create project → Enable YouTube Data API
  3. Create API Key
  4. export YOUTUBE_API_KEY=xxx

📸 Instagram

  1. Meta for Developers
  2. Create app → Add Instagram Basic Display
  3. Get Access Token
  4. export INSTAGRAM_TOKEN=xxx

🐦 Twitter/X

  1. Twitter Developer
  2. Create app → Get API Keys
  3. Copy Key & Secret
  4. export X_API_KEY=xxx

🤖 OpenRouter (AI)

  1. openrouter.ai/keys
  2. Create free account
  3. Generate API key
  4. export OPENROUTER_API_KEY=xxx

💡 Tip: Add these exports to your ~/.bashrc or ~/.zshrc to persist them

❓ Troubleshooting

"Claude can't find my skills"

→ Restart Claude completely (quit and reopen)

"API key not working"

→ Make sure you ran: export KEY=value in the same terminal

"Permission denied"

→ Check that your API key has the right permissions enabled (e.g., YouTube Data API)
