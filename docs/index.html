Simple setup

Choose your preferred method. Works with Claude Desktop, Claude Code, or Kai CLI.

1

Install kai-skills

No Kai CLI required - just this helper

npm install -g kai-skills
2

Install skills

Downloads all Kai skills to ~/.kai/skills

kai-skills install
3

Add to Claude

Configures all skills automatically

kai-skills add all
4

Start using

Restart Claude and ask: "Get my YouTube stats"

1

Install skills

Clone the skills repository

git clone https://github.com/tmoreton/kai-skills ~/.kai/skills
2

Add to Claude Desktop

Manually add each skill via MCP settings

Command+Shift+P → Add MCP Server
3

Configure

For each skill, add:

Command: node
Args: ~/.kai/skills/youtube/handler.js
4

Start using

Restart Claude and ask: "Get my YouTube stats"
