API Key Setup

Get your free API keys. We have three ways to help:

Easiest: Interactive Wizard

Walks you through step by step
npx kai-api-setup
View web guide →

One-Click OAuth

No API keys needed (coming soon)
npx kai-auth connect youtube
YouTube
  1. Google Cloud Console
  2. Create project → Enable YouTube Data API
  3. Create API Key
  4. export YOUTUBE_API_KEY=xxx
Instagram
  1. Meta for Developers
  2. Create app → Add Instagram Basic Display
  3. Get Access Token
  4. export INSTAGRAM_TOKEN=xxx
Twitter/X
  1. Twitter Developer
  2. Create app → Get API Keys
  3. Copy Key and Secret
  4. export X_API_KEY=xxx
OpenRouter (AI)
  1. openrouter.ai/keys
  2. Create free account
  3. Generate API key
  4. export OPENROUTER_API_KEY=xxx
Tip: Add these exports to your ~/.bashrc or ~/.zshrc file so they persist across terminal sessions.

Troubleshooting

?
Claude cannot find my skills

Restart Claude completely. Quit the application and reopen it.

?
API key not working

Make sure you exported the key in the same terminal window: export KEY=value

?
Permission denied errors

Check that your API key has the correct permissions enabled in the developer console.
