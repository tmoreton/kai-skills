API Key Setup

Get your API keys from social media platforms

YT
YouTube
Easy - 5 min
1

Go to Google Cloud Console

Click the button below to open the console

Open Google Cloud Console →
2

Create a new project

Click "Select a project" → "New Project" → Name it "My Social Analytics"

Screenshot: Project selector dropdown
3

Enable YouTube API

Search "YouTube Data API v3" and click "Enable"

Screenshot: API Library search
4

Create API Key

Go to "Credentials" → "Create Credentials" → "API Key"

Screenshot: Create credentials button
5

Copy your key

A popup will show your key. Copy it and run this in your terminal:

export YOUTUBE_API_KEY=your_key_here

Stuck?

Use the interactive wizard: npx kai-api-setup

IG
Instagram
Medium - 10 min
1

Go to Meta for Developers

Open Meta Developer Portal →
2

Create an app

Click "Create App" → Select "Other" → Name it "Social Analytics"

3

Add Instagram Basic Display

In the left sidebar, click "Add Product" → Find "Instagram Basic Display" → Click "Set Up"

4

Generate token

Go to "Basic Display" → "User Token Generator" → Click "Generate Token"

export INSTAGRAM_TOKEN=your_token_here
OR
OpenRouter (AI)
Easiest - 2 min
1

Sign up on OpenRouter

Go to OpenRouter →
2

Create API Key

Click "Create API Key" → Name it "Kai" → Copy the key

export OPENROUTER_API_KEY=your_key_here
